# 🚀 Portfolio de Thomas - Graphiste

Ce projet consiste en l'intégration d'un portfolio professionnel et responsive pour un graphiste indépendant.

## 🔗 Liens du projet
- **Site en ligne (Alwaysdata) :** [https://hermann.alwaysdata.net/]
- **Dépôt GitHub :** [https://github.com/LettreH/portfolio-thomas.git]

## 🛠️ Caractéristiques du projet

### 📱 Responsive Design
- Mise en page fluide réalisée avec **Flexbox** et **CSS Grid**.
- Site entièrement adapté aux mobiles, tablettes et ordinateurs.
- Validation de l'affichage avec un **zoom à 200%** pour garantir la lisibilité.

### 🏗️ Structure Sémantique
- Utilisation des balises **HTML5** (`<header>`, `<nav>`, `<section>`, `<footer>`) pour un code propre et mieux référencé.
- Navigation fluide entre les différentes sections du portfolio.

### 🖼️ Optimisation des visuels
- Utilisation du format **WebP** pour les images afin d'accélérer le chargement des pages tout en gardant une excellente qualité visuelle.
- Présentation claire des travaux de graphisme de Thomas.

## 📁 Organisation des fichiers
- `index.html` : Contenu et structure sémantique du site.
- `style.css` : Feuilles de styles et règles responsive.
- `/images` : Dossier contenant les assets et réalisations de Thomas.
- `favicon.ico` : Icône personnalisée pour l'onglet du navigateur.
